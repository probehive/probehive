# ProbeHive Brand Assets

This directory is the authoritative source for ProbeHive brand assets. The SVG files are the masters; raster files are exports derived from them.

| File | Purpose |
| --- | --- |
| `probehive-logo.svg` | Master logo, 1024×1024, square dark background |
| `probehive-logo.png` | 1024×1024 export for avatars and platforms that require raster images |

Derived artifacts such as favicons or application icons are generated from these masters and live with the surface that ships them (for example the web application's public assets), not here.
